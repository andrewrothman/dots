- ripgrep

- gh

### Node.js / React.js / JavaScript / TypeScript

- n
- deno

## Setup

```bash
git clone git@github.com:andrewrothman/dots.git
cd dots

stow nvim -t ~
stow alacritty -t 
stow tmux -t ~
stow efm-langserver -t ~

# on mac
stow karabiner -t ~
```
